Options can be **set** or **unset** using Artisan commands or using the Administration.

<Aside type="note">
When used with modules, the group represents the module identifier.
</Aside>
## List All Options

To view all registered options, run the command:

```bash
php artisan options:list {groups?*}
```

* `{groups?*}` – *optional*: specify one or more groups to filter the list.
  If omitted, all options from all groups will be displayed.


```text
 ┌──────────────────────┬─────────────────────────┬───────────────────────────────┬───────┬────────────────────────────────┬─────────┐
 │ Group                │ Name                    │ Key                           │ Value │ Default                        │ Type    │
 ├──────────────────────┼─────────────────────────┼───────────────────────────────┼───────┼────────────────────────────────┼─────────┤
 │ core::administration │ Enable Modules Manager  │ enable-modules-manager        │ null  │ true                           │ Boolean │
 │ core::administration │ Brand Name              │ brand-name                    │ null  │ "Laravel"                      │ String  │
 │ core::administration │ Single-Page-Application │ spa                           │ null  │ true                           │ Boolean │
 │ core::administration │ Top Navigation          │ top-navigation                │ null  │ false                          │ Boolean │
 │ core::mcp-server     │ Server name             │ name                          │ null  │ "Laravel internal MCP Server"  │ String  │
 │ core::mcp-server     │ Server version          │ version                       │ null  │ "dev-main"                     │ String  │
 └──────────────────────┴─────────────────────────┴───────────────────────────────┴───────┴────────────────────────────────┴─────────┘
```

## Set a Specific Option

To set an option, run:

<Aside type="tip">You can omit arguments to use the interactive prompt.</Aside>

```bash
php artisan options:set {group} {key} {value}
```
```text
 ┌ Which group do you want to update? ────────────────────────┐
 │ core::administration                                       │
 └────────────────────────────────────────────────────────────┘

 ┌ Which key do you want to modify? ──────────────────────────┐
 │ brand-name                                                 │
 └────────────────────────────────────────────────────────────┘

 ┌ Enter the new value for the selected key: ─────────────────┐
 │                                                            │
 └────────────────────────────────────────────────────────────┘
```

## Unset a Specific Option

<Aside type="note">
When an option is unset, the system will revert to the **default value** provided by the schema.
</Aside>

To unset an option:

<Aside type="tip">You can omit arguments to use the interactive prompt.</Aside>

```bash
php artisan options:unset {group} {keys*}
```
```text
 ┌ Which group do you want to update? ────────────────────────┐
 │ core::administration                                       │
 └────────────────────────────────────────────────────────────┘

 ┌ Select the key(s) to unset ────────────────────────────────┐
 │   ◼ enable-modules-manager                                 │
 │ › ◼ brand-name                                             │
 │   ◻ spa                                                    │
 │   ◻ top-navigation                                         │
 └────────────────────────────────────────────────────────────┘
```
